## 關於 TripWay
TripWay 是一個規劃旅遊行程的網站，您可以建立專屬行程，在行程裡設定旅遊天數，並新增這一天要去的景點資訊。
更棒的是您可以邀請旅伴一起共同參與旅程規劃，讓整個旅程規劃更加條理，讓您的旅程的時間充滿趣味與安心！
