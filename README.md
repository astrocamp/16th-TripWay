# TripWay 趣味

## HOW TO START
- `poetry shell`enter or activate the virtual environment.<br>
- `poetry install`<br>
- `npm run dev`to start the frontend server.<br>
- `node install`<br>
- `npm install`<br>
## MIGRATION TO MYSQL
- python manage.py `makemigrations`<br>
- python manage.py `migrate`<br>
